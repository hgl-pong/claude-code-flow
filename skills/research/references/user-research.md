# User Research

Rigorous research beats assumptions. Plan the method, script the sessions, know what you're looking for before you start.

## Research Methods

| Method | Best For | Sample Size | Time |
|--------|----------|-------------|------|
| Interviews | Motivations, workflows, depth | 5-8 per segment | 45-60 min each |
| Usability Testing | Task completion, friction points | 5-8 (finds 85% of issues) | 30-60 min each |
| Surveys | Quantify patterns, large samples | 100+ for significance | 5-10 min each |
| Card Sorting | IA, mental models, categorization | 15-20 | 15-30 min each |
| Diary Studies | Behavior over time, longitudinal | 10-15 | 1-2 weeks |
| A/B Testing | Compare variants, measure impact | 1,000+ per variant | 1-4 weeks |

## Interview Guide Structure

```markdown
## Warm-up (5 min)
- Build rapport, explain purpose, get consent
- "Tell me about your role..."

## Context (10 min)
- Understand current workflow and environment
- "Walk me through how you currently..."

## Deep Dive (20 min)
- Explore specific behaviors, pain points, needs
- "What's the hardest part about..."
- "Can you show me how you..."
- "What would make this easier?"

## Reaction (10 min)
- Test assumptions, present concepts if applicable
- "How would you expect this to work?"
- "What concerns you about this approach?"

## Wrap-up (5 min)
- Final thoughts, anything missed
- "Is there anything else you'd like to share?"
```

## Analysis Frameworks

| Framework | Use When | Output |
|-----------|----------|--------|
| Affinity Mapping | Multiple participants, find themes | Clustered themes with quotes |
| Impact/Effort Matrix | Prioritize findings | 2x2 grid: quick wins vs. big bets |
| Journey Mapping | Multi-step workflows | Stages + emotions + pain points |
| Jobs to Be Done | Understand motivation | "When [situation], I want to [motivation], so I can [outcome]" |

## Deliverables

- **Research Plan**: Objective, research questions, method, participants, timeline
- **Interview Guide**: Scripted questions per section above
- **Synthesis Report**: Key findings, supporting quotes, recommendations
- **Highlight Reel**: 5-10 clips illustrating top findings
